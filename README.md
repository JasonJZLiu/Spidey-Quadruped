# Spidey-Quadruped

## Spidey: A 12 DOF Sprawling-Type Quadruped Robot

### Hardware:
- Raspberry Pi 4
- Adafruit 16-Channel 12-bit PWM/Servo HAT
- MG90 Servos

### Software:
- Controlled via trajectory interpolation and inverse kinematics
- Simulated in Nvidia Isaac Sim and Drake
